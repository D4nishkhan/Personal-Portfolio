# Danish Khan - Backend Architect Portfolio

A premium, handcrafted portfolio website that breaks conventional templates and creates an immersive storytelling experience.

## 🎯 Design Philosophy

This portfolio is designed to feel like an **experience**, not a resume page. It features:

- **Non-linear layout** with scroll-based reveals and layered storytelling
- **Unique color palette** - soft peach, lavender, and warm grays
- **Interactive animations** using Framer Motion and GSAP
- **Particle network background** representing microservices architecture
- **Glassmorphism and soft shadows** for a modern aesthetic

## ✨ Features

### Dynamic Hero Section
- Animated letter-by-letter name reveal
- Custom cursor follower with gradient glow
- Floating availability indicator
- Real-time stats card

### Philosophical Approach
- Storytelling instead of bullet points
- Interactive value cards (Architecture, Scalability, Performance, Reliability)
- Visual metaphors for backend development concepts

### Journey Timeline
- Alternating layout with gradient markers
- Milestone cards with hover effects
- Visual representation of growth from 2021-2025

### Tech Ecosystem
- Interactive technology cards with proficiency levels
- Animated progress bars
- "Currently Exploring" section showcasing continuous learning

### Project Showcases
- Expandable project cards
- Key highlights with animated reveals
- Tech stack tags with glassmorphism

### Animated Background
- Canvas-based particle network
- Represents microservices and API connections
- Dynamic node connections with distance-based opacity

## 🛠 Tech Stack

- **React** - UI framework
- **TypeScript** - Type safety
- **Vite** - Build tool
- **Tailwind CSS** - Styling
- **Framer Motion** - Smooth animations
- **GSAP** - Scroll-triggered animations
- **Three.js** - 3D capabilities (ready for future enhancements)

## 🎨 Color Palette

- **Peach**: `#FFD5C2` to `#FF8F62` - Warm, inviting accent
- **Lavender**: `#E6E6FA` to `#9B86C7` - Calm, sophisticated accent
- **Warm Gray**: `#FAF9F7` to `#2D2A26` - Neutral base with warmth

## 🚀 Performance

- Optimized animations with RAF (requestAnimationFrame)
- Lazy-loaded scroll animations
- Efficient particle system
- Responsive design for all devices

## 📱 Responsive Design

Fully responsive from mobile to 4K displays with:
- Adaptive grid layouts
- Mobile-optimized animations
- Touch-friendly interactions

## 🎭 Notable Design Decisions

1. **No cliché sections** - Avoided "About → Skills → Projects → Contact" structure
2. **Storytelling over listing** - Content integrated into narrative blocks
3. **Meaningful motion** - Every animation serves a purpose
4. **Backend theme** - Visual metaphors for systems, flows, and connections
5. **Premium feel** - Attention to micro-interactions and details

## 💡 Customization

To customize for your own use:

1. Update personal information in component files
2. Modify color palette in `tailwind.config.js`
3. Replace project details in `Projects.tsx`
4. Adjust tech stack in `TechEcosystem.tsx`
5. Update journey milestones in `Journey.tsx`
